- Dark luxury aesthetic with gold accents

- Real Unsplash images for interior design portfolio

## Deployment

Push all 4 files to the root of your `interiordesignerinkarachi.github.io` repository. GitHub Pages will serve `index.html` automatically.

```
interiordesignerinkarachi.github.io/
├── index.html
├── robots.txt
├── sitemap.xml
└── README.md
```

After deployment, submit your sitemap to **Google Search Console**:
`https://interiordesignerinkarachi.github.io/sitemap.xml`
